## 回顾页 (Review) — 温暖插画风

### 整体布局

```
┌──────────────────────────────┐
│      你的感恩之路 🌷          │
│    🔥 连续感恩第 7 天         │
├──────────────────────────────┤
│  🔍 搜索日记内容、心情...    │  搜索栏
├──────────────────────────────┤
│  ▸ 📅 2026年6月    本月 ▾    │  折叠月份标题
├──────────────────────────────┤
│  ┌────────┐  ┌────────┐     │
│  │ 😊 家人 │  │ 🥰 自然 │     │  瀑布流双列
│  │ 06-04  │  │ 06-03  │     │
│  │ 今天.. │  │ 看到.. │     │
│  │  ● ○ ○ │  │  ● ● ○ │     │
│  └────────┘  └────────┘     │
├──────────────────────────────┤
│  ▸ 📅 2026年5月    3篇  ▾    │
│  ┌────────┐  ┌────────┐     │
│  │ ...    │  │ ...    │     │
└──────────────────────────────┘
```

### 设计决策

1. **搜索栏**：全宽圆角输入框，支持搜索内容、心情名、分类名、日期
2. **月份折叠**：点击展开/收起，chevron 旋转动画，当前月份默认展开
3. **瀑布流**：`columns-2` 双列不规则排列
4. **DiaryCard 温暖插画风**：
   - 根据心情匹配柔和渐变背景（感恩=橙→黄，温暖=粉→橙，平静=紫→蓝…）
   - emoji 放在白色小圆角框内
   - 底部三个渐隐彩色圆点作为插画装饰
   - 规整 `rounded-2xl`，不歪斜不异形
   - 入场弹性动画，hover 上浮，tap 缩小
5. **空状态**：无日记时显示引导文案

## 分析页 (Analysis)

### 布局

```
┌──────────────────────────────┐
│      你的感恩分析 📊          │
├──────────────────────────────┤
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌──┐│
│  │ 42  │ │ 12  │ │  7  │ │21││
│  │总篇数│ │本月 │ │连续 │ │最长│
│  └─────┘ └─────┘ └─────┘ └──┘│
├──────────────────────────────┤
│  情绪分布        🍩 环形图    │
├──────────────────────────────┤
│  感恩分类排行  ████████ 进度条│
├──────────────────────────────┤
│  🤖 AI 月度回顾  🔄 可刷新   │
└──────────────────────────────┘
```

### 各模块详情

#### 统计卡片
- 4 个指标并排，`grid-cols-4`
- 白底圆角卡片，大数字 + 小标签

#### 情绪分布图
- recharts PieChart 环形图 + 右侧图例
- 每种心情对应暖色系颜色

#### 感恩分类排行
- TOP 5 横向进度条，显示排名 + 分类名 + 进度 + 次数

#### AI 月度回顾
- 调用 DeepSeek API（`MONTHLY_SUMMARY_PROMPT`）
- 生成后缓存到 IndexedDB settings（key = `ai-summary-YYYY-MM`）
- 未配置 API Key 时显示引导文案
- 本月无日记时不显示
- 加载中显示 spinner，失败显示重试提示
- 手动刷新按钮清除缓存重新生成
